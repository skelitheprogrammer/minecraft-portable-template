# ⚠️ Важное предупреждение

> **Без VPN загрузка файлов с серверов невозможна!**  
> Убедитесь, что VPN включен **до** начала установки.

---

# 🚀 Установка клиента

Нажмите правой кнопкой мыши по ссылке ниже и выберите **«Сохранить ссылку как…»**

### 📥 [wizard installer](https://raw.githubusercontent.com/skelitheprogrammer/minecraft-portable-template/refs/heads/master/wizard.ps1)

**Альтернативный способ (через консоль):**  
Выполните эту команду в выбранной папке:

```cmd
curl -L -o wizard.bat https://raw.githubusercontent.com/skelitheprogrammer/minecraft-portable-template/master/wizard.bat && wizard.bat
```


# :inbox_tray: Процесс установки
1. Скачивание
    Либо из репозитория
    Либо по прикреплённой ссылке из Discord

2. Запустите скрипт
Он подготовит файлы и автоматически запустит create-profile.ps1.

3. Настройка профиля

    Выберите название профиля
    
    Выберите имя пользователя
    
    Впишите версию неофорджа
    
    Впишите желаемые аргументы (прочитайте подсказки в консоли)
    или примите дефолтные — 8 ГБ ОЗУ

4. Запуск
     Запустите скрипт `portablemc-run.ps1`

5. Ожидание
    Дождитесь полной установки.
### :warning::warning::warning: Если не качается — используйте VPN.:warning::warning::warning:
### Скачивание идёт с зарубежных серверов.

6. Зайти на сервер по IP

    По просьбе сервера вставить ключ-сертификат
    
    Подтвердить
    
    Принять загрузку модов.
7. Перезапустить клиент
8. Зайти на севрер
9. ???
10. Играц 

# Будущее использование
